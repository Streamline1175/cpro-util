How auto-discovery works

cpro-util tries to resolve three mDNS hostnames: centerpiece.local, finalmouse-centerpiece.local, finalmouse.local. Your OS must support mDNS (macOS has it built in; Windows needs Bonjour from iTunes; Linux needs avahi-daemon running). The keyboard and your computer must be on the same Wi-Fi network.

Finding the keyboard IP manually

  1. Open your router's admin panel (usually 192.168.1.1 or 192.168.0.1)
  2. Find the device list / DHCP leases page
  3. Look for a device named centerpiece, finalmouse, or with an ARM MAC prefix
  4. Copy its IP address and paste it in the "Manual host" field above

Firewall & port notes

The keyboard listens on port 8080. Make sure your local firewall allows outbound connections to port 8080 on the local subnet. No authentication is required — the endpoint is community-reverse-engineered and not officially documented by Finalmouse.
