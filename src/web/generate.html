      How this works ▸
    
    

      

        1. Describe the visual effect you want in plain English.
        Claude (Anthropic) generates a complete setup_interactive.py that
        you run once inside the UE 4.27.2 editor console — it scaffolds all the Blueprints,
        Niagara particle systems, and levels for you automatically.
      

      

        2. Author Open the generated UE project, run the Python script,
        then tweak the Niagara system in the editor to match the generated concept.
        The script logs exact module settings to the UE output log.
      

      

        3. Cook & upload from your terminal:
      

      
cpro ue pak cook  ./my-skin-project
cpro ue pak upload dist/skin.pak --slot 0

      

        VS Code users: open the Command Palette
        (Ctrl+Shift+P) and run cpro: Generate Interactive Skin
        to launch this workflow directly from the editor.
